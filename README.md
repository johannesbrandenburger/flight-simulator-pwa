## Keyboard Shortcuts in Flight Simulator

| Key | Function |
| --- | --- |
| P   | Pause |
| I   | Invert Controls |
| F   | Exit Flight Simulator and Return to Flight School Scene |
| J   | Toggle FPS Display Visibility |
